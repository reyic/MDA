PRADROID

Le presentamos Pradroid, en esta web de iniciacion de aplicacion. Nuestro proyecto, consta de una parte web y una aplicacion android para smartphone.

Nuestra plataforma, puede llegar a ser muy interesante para los distintos centros docentes que componen el mundo. Por supuesto hablamos de institos, colegios y universidad.
Nuestra aplicacion web se compone de las siguientes caracteristicas.

  • Registro de profesores y alumnos.
  • Creacion de las distintas asignaturas que se imparta el centro.
  • La web cuenta con funcionalidades como:
    • Creacion de foros de discusion
    • Chat interno, con el cual los alumnos y los profesores pueden estar en contacto.
    • Notificaciones en el sistema.
    • Consulta de calificaciones.
    • Subida de material.
    • Entrega de trabajos.
  • Nuestra aplicacion para smartphone tiene las siguientes funcionalidades.
    • Recibir notificaciones desde el sistema web.
    • Enviar mensajes.
    • Consultar materiales.
  • Si eres profesor, desde tu smartphone podras:
    • Responder dudas a tus alumnos.
    • Crear entregas de trabajos y/o practicas.
    • Crear temas en el foro y administrarlo.
